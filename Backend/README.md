## Flask Installation and Running Instructions
### Installing the Flask Server
1. Make sure that you have Python 3.9.5 or above by running the following in the command line: `python --version`
2. Clone/download the repository to your local machine
3. Using the command line, go to the main directory: `cd YOURPATH\backend`
4. Create a python virual environment: `python -m venv venv`
5. Activate the virtual environment: `./venv\Scripts\activate`
6. Install the required depedencies: `pip install -r requirements.txt`
7. Run the Flask server: `python manage.py runserver`

Note: these instructions are on windows 10 machines. It should be similar on other systems as well.

### Running the Flask Server
1. Using the command line, go to the main directory: `cd YOURPATH\backend`
2. Activate the virtual environment: `./venv\Scripts\activate`
3. Run the Flask server: `python manage.py runserver`

### Updating the Required Dependencies
1. Using the command line, go to the main backend directory: `cd YOURPATH\backend`
2. Install the required depedencies: `pip install -r requirements.txt`
4. Run the Flask server
